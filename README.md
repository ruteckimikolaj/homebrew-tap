# homebrew-tap

Homebrew formulae for tools by [@ruteckimikolaj](https://github.com/ruteckimikolaj).

## Usage

```shell
brew tap ruteckimikolaj/tap
```

Then install any formula:

```shell
brew install ruteckimikolaj/tap/pomodorust
brew install ruteckimikolaj/tap/gcp-snap-crab
```

## Formulae

| Formula | Description |
| ------- | ----------- |
| [pomodorust](Formula/pomodorust.rb) | Minimalist terminal-based Pomodoro timer |
| [gcp-snap-crab](Formula/gcp-snap-crab.rb) | Terminal-based GCP backup and restore tool |
